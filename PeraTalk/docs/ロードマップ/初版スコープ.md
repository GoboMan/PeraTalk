# 初版スコープ（Phase 1）

[← README に戻る](../../README.md)

初回リリース（Phase 1）で**含むもの／含まないもの**を [機能一覧](../機能/機能一覧.md) へのリンク集として整理する。**機能の詳細仕様はここには持たない**（二重記述を避けるため、各機能ファイルを正とする）。

## Phase 1 に含む

| ID | 機能名 | 詳細 |
|----|--------|------|
| `CONV` | 会話（Self／AI、A／B 入力） | [会話](../機能/会話.md) |
| `PERSONA` | 会話ペルソナ（6 パターン）と端末 TTS | [会話-ペルソナとTTS](../機能/会話-ペルソナとTTS.md) |
| `PDF` | 会話 PDF 出力 | [会話 §1](../機能/会話.md#1-仕様モード一覧) |
| `VOC-LIST` | 単語一覧 | [単語帳](../機能/単語帳.md) |
| `VOC-BOOKMARK` | ブックマーク | [単語帳](../機能/単語帳.md) |
| `VOC-TAG` | タグ付け | [単語帳](../機能/単語帳.md) |
| `VOC-LISTEN` | 単語の読み上げ | [単語帳](../機能/単語帳.md) |
| `LOG` | 学習ログ | [学習ログ](../機能/学習ログ.md) |
| `SET` | 設定とアカウント | [設定とアカウント](../機能/設定とアカウント.md) |

## リリース基準（暫定）

- Phase 1 のすべての機能で**受け入れ条件**（各機能ファイルの該当節）を満たすこと。
- [収益とサブスク](../運営/収益とサブスク.md) の **プラン別機能一覧**（無料／プラス／プロ。店頭価格は **プラス・プロ**列見出しセル内の月額・年額）どおりに StoreKit 上で課金が動作すること。
- 認証経路（[設定とアカウント](../機能/設定とアカウント.md)）が一通り疎通していること。

> 受け入れ条件の詳細・テスト方針は実装フェーズで各機能ファイルに追記する。

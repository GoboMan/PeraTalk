# 機能一覧

[← README に戻る](../../README.md)

**画面のことは一切考慮せず**、ユーザーが得られる**能力（機能）単位**でまとめたカタログ。詳細仕様は各機能ファイルを、画面（情報設計）との対応は [画面一覧](画面一覧.md) を参照。

- **`ID`**：Issue／コードのコメント／ロードマップ／[画面一覧](画面一覧.md) から参照する**安定した識別子**。
- **`状態`**：設計中／実装中／完了。
- **`フェーズ`**：[初版スコープ](../ロードマップ/初版スコープ.md) を正とする（Phase1 ／ 後フェーズ）。

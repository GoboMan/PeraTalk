# PeraTalk

iOS 向け**英会話**学習アプリ。**ボキャブラリーブック**と**会話（Self／AI）**を中心に、**アウトプット → フィードバック → インプット**のループで「自分の語彙を文として使える力」と「場面に応じた表現の引き出し」を伸ばす。

> プロダクトの背景・目的・差別化コンセプトの本文は [docs/概要/ビジョンと目的](docs/概要/ビジョンと目的.md) を参照。

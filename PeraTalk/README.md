# PeraTalk

iOS 向け**英会話**学習アプリ。**主な想定は英語の中上級者**——会話で**積極的にアウトプット**してもらい、その中でフィードバックと振り返りを回す前提である。**ボキャブラリーブック**と**会話（Self／AI）**を中心に、**アウトプット → フィードバック → インプット**のループで「自分の語彙を文として使える力」と「場面に応じた表現の引き出し」を伸ばす。

**対応 OS**：**Phase 1 は iOS 26+**（判断理由・運用ポリシーの SSOT は [初版スコープ §対応 OS](docs/ロードマップ/初版スコープ.md#対応-os)）。Apple Intelligence（オンデバイスモデル）を IPA 生成・単語帳 AI ドラフト・RAG スキップ判定で活用するため、**最低 OS を iOS 26 に揃え、フォールバック分岐を最小化**する設計。

> プロダクトの背景・目的・差別化コンセプトの本文は [docs/概要/ビジョンと目的](docs/概要/ビジョンと目的.md) を参照。
